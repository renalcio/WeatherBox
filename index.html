Weather Box

Previsão do tempo no seu site.

Weather widget for your web site.

Este plugin adiciona uma caixa com a previsão do tempo em qualquer página HTML.
This plugin insert a box with the weather in any HTML page

Requerimentos Requirements

  • JQuery 2x +
  • Skycons *
  • Line Icons *

* Itens já inclusos no download. / * Include on download zip

Opções Options

Opção / Option Padrão / Default Descrição / Description
city "auto" String - Cidade que deseja prever o tempo
String - Town to predict the weather
animate true Boolean - Ativar animação
Boolean - Enable animation
lang "pt" String - Idioma das frases
String - Language
unit "c" Char - Unidade meteorológica.
Char - Weather unit
Celcius, Fahrenheit, Kelvin
geolocation true Boolean - Usar geolocalização do navegador
Boolean - Use browser geolocation
lat 0 Float - Latitude, defina como 0 ou não defina para usar a cidade
Float - Latitude, set as 0 or not set to use the city
lon 0 Float - Longitude, defina como 0 ou não defina para usar a cidade
Float - Longitude, set as 0 or not set to use the city
